# SerenFit

An adaptive AI-powered fitness coaching app built with React Native (Expo) and a Node.js backend.

## About

SerenFit generates personalised daily meal and workout plans, tracks your progress, and lets you chat with an AI coach (Alex) that adapts your plan in real time based on how your day is going.

## Tech Stack

- **Mobile** — React Native (Expo SDK 54), NativeWind, PlusJakartaSans
- **Backend** — Node.js, Express, Prisma, PostgreSQL
- **AI** — Backboard SDK (Claude-powered agents)

## Features

- Personalised onboarding questionnaire
- AI-generated weekly meal and workout plans
- Swipeable logging cards with haptic feedback
- AI coach chat with real-time plan adjustments
- Week-by-week history navigation
